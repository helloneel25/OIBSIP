# BMI Calculator

## Description

A Python command-line application that calculates Body Mass Index (BMI) based on the user's weight and height.

## Features

- Calculate BMI
- BMI classification
- Input validation
- Health suggestions

## Technologies Used

- Python 3

## How to Run

```bash
python BMI_calculator.py
```

## BMI Categories

- Underweight
- Healthy Weight
- Overweight
- Obese
